# Публикация в канал

Как работает поле «Канал» и кто может публиковать посты через @RichMessages_bot.

## Два действия в редакторе

| Кнопка | Куда уходит пост | Кто может |
|--------|------------------|-----------|
| **Себе** | Личка пользователя с ботом | Любой, кто открыл бота и нажал `/start` |
| **В канал** | Указанный `@channel` | Только **админ или владелец** этого канала |

## Поле «Канал @channel»

Пользователь вводит `@username` канала или numeric id (`-100…`).

Если поле пустое — сервер может подставить канал по умолчанию из настроек (подсказка в UI).

**Права проверяются на сервере**, не во фронте.

## Кто может публиковать

| Роль пользователя в канале | Публикация |
|--------------------------|------------|
| Владелец | ✅ |
| Админ с правом постить | ✅ |
| Админ без права постить | ❌ |
| Подписчик / не в канале | ❌ |

## Два условия для публикации

1. **Пользователь** — владелец или админ канала.
2. **Бот @RichMessages_bot** — администратор канала с правом публиковать сообщения.

## Типичный сценарий

1. Создать канал `@my_blog`.
2. Добавить @RichMessages_bot **администратором** канала.
3. Открыть бота → `/start` → Mini App.
4. Создать пост → ввести `@my_blog` → **В канал**.

## Рекомендации оператору

- **Не добавляйте бота админом** в каналы, куда не хотите давать publish-пользователям.
- Канал по умолчанию в настройках — только подсказка; публиковать может только его админ.
- Превью «Себе» не требует прав канала.
